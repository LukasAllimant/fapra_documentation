feature_extraction module¶

Module with different functions for the feature extraciton

feature_extraction.add_all_features_to_df(df)¶

gets a pandas dataframe with the filenames as colum. calculates all possible features and adds them to the corresponding colums in the pandas dataframe

Args:

df (pandas dataframe): dataframe with the filenames and path in a colum

Returns:

pandas dataframe: same dataframe but the colums for all features are filled out

feature_extraction.calc_labels(path_origin='Data/original_dataset/', path_goal='Data/features/')¶

reads all filenames in a given folder, reads the labels from a soundfile, writes them into an array and saves the array as pickle file

Args:

path_origin (String, optional): Path where the wav fales stored. Defaults to PATH_ORIGINAL_DATASET. path_goal (String, optional): Path where the pickle file shall be stored. Defaults to PATH_FEATURES.

Returns:

None: None
