deep_learning module¶

Module for for deep learning evaluation

deep_learning.create_dnn_model(learn_rate=0.001, activation='relu', kernel_initializer='glorot_uniform', dropout_rate=0.3, weight_constraint=10, neurons_l1=24, neurons_l2=48, neurons_l3=24, input_size=12)¶

Creates keras tensorflow dnn model, different Hyperparameters can be passed as parameters, the structure of the dnn is a keras sequential model with 3 hidden dense layers an one dropout layer

Args:

learn_rate (float, optional): learning rate of the Adam optimizer. Defaults to 0.001. activation (str, optional): activation function used in all hidden layers. Defaults to ‘relu’. kernel_initializer (str, optional): kernel initializer used to initialize the wheigts. Defaults to ‘glorot_uniform’. dropout_rate (float, optional): dropout rate for the dropout layer. Defaults to 0.3. weight_constraint (int, optional): weight constraints. Defaults to 10. neurons_l1 (int, optional): number of neurons in the first hidden layer. Defaults to 24. neurons_l2 (int, optional): number of neurons in the second hidden layer. Defaults to 48. neurons_l3 (int, optional): number of neurons in the third hidden layer. Defaults to 24. input_size (int, optional): Size of the input feature vector. Defaults to 12.

Returns:

keras sequential model: keras sequential model with given hyperparameters
