standard_machine_learning module¶

Evaluation module for standard machine learning classifiers Nested cross validation for evaluation and hyperparametertuning

standard_machine_learning.nested_cv_evaluation(model, param_space, scaler, feature_name)¶

Perform nested cross validation with hyperparametertuning and model evaluation for a sklearn classifier, the hyperparametertuning is done by random search

model (sklearn classifier):

A sklearn classifier object that should be evaluated with nested cross validation

param_space (dict):

dictonary with hyperparameters and their possible distributions

scaler (sklearn preprocessing scaler):

scaler which is used to scale the datasets, for standard machine learning the StandardScaler is recommended

feature_name (str):

name of the feature set to use
